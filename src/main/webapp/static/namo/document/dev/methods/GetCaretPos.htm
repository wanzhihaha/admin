Remarks

메소드는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

GetCaretPos로 구한 캐럿의 위치 값은 SetCaretPos에서 사용할 수 있는 값으로 이 값의 절대 값은 아무런 의미를 가지고 있지 않습니다.
