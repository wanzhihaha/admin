IsDirty

편집 중인 문서의 내용이 변경되었는지를 확인합니다.

 

Syntax

    HRESULT IsDirty();

Parameters

없음

 

Return Values

TRUE

문서의 내용이 변경되었음

FALSE

문서의 내용이 변경되지 않았음

 

Remarks

메소드는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

 

Sample Codes

JavaScript

var object = new NamoSE("test");
object.EditorStart();
function OnInitCompleted(){
   var info = object.IsDirty();
   if(info)
   {
       alert("문서의 내용이 바뀌었습니다.");
   }
   else
   {
       alert("문서의 내용이 바뀌지 않았습니다.");
   }
}
